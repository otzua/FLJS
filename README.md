# FLJS - Japanese Learning App (100% vibe coded)

FLJS is a minimalist and powerful Android application designed to help users master Japanese characters (Hiragana and Katakana) and practice reading with real-world passages.

## Features
- **Quiz Mode:** Practice Hiragana, Katakana, or a mixture of both.
- **Reading Practice:** Read real-world passages from NHK Web Easy and Aozora Bunko.
- **Minimalist Design:** Fast, clean, and focused on learning.
- **Customizable:** Adjust the number of questions and character types.
